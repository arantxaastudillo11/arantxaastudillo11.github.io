Sketch 7

For sketch seven, I chose to research "spring" and "curve", which were a bit challenging for me. I was able to understand spring, and able to change its movement, however when I changed the object from ellipse to anything else, such as rectangle or an image, the object would not spring properly or not spring at all. I used the curve as a guide to let the user know how to get the spring into the basketball net for a little game.

Project 1

The algorithm I used in sketch 7 was spring. In this sketch, I reused this algorithm to create the six black circles that can be clicked on and dragged, then once the mouse is release, the circles will spring right back to their original positions. The custom function I used is astudillogrowingellipse (); and this function made the four black circles that begin in the center of the window and move outward while growing wider and wider. Lastly, I used saveFrame to record 600 jpg of my code and made an MP4 using Adobe Media Encoder.
